B. INSTALLATION

1. Unzip the blogMX.zip file in to a directory on your server.

2. Folders listing:

a. cryptadmin: contains all admin pages. Do not make any changes in the source codes of the files in this folder if you are not comfortable with ASP. If you want to redesign or change the color/layout of the admin pages, you should use an editor of your choice to do it (Dreamweaver MX or Dreamweaver UltraDev 4 are preferred)

b. includes: contains all included files, viz:

  • allscripts.js - required javascripts
  • calendar.asp - includes the calendar in blog
  • links.asp - includes links in blog
  • logo.gif - logo for blog > change this file as required
  • menu.asp - the site navigation menu > change this as required
  • month.asp - includes month links in blog
  • recent.asp - includes recent blogs in permalink file
  • search.asp - includes search box in blog
  • uploaded_ images directory: this folder will contain any uploaded images. It must have write permissions

c. Connections: this folder contains a file called dreamConn.asp; read the '3. Connecting to the database:' below to learn more about this file.

d. Database: contains dbblogMX.mdb; read the '3. Connecting to the database:' below to learn more about this file.

e. DO NOT MAKE ANY CHANGES to the files unless you're comfortable with coding in ASP

3. Connecting to the database:

- blogMX.zip contains a folder named Connections. There is an ASP file in the folder called dreamConn.asp

- After you have placed all files and the database file into their directory, open dreamConn.asp in UltraDev, Dreamweaver MX, Notepad, etc. Below are the codes inside this file:

- If you want to use DSN, create a DSN call dsnBlogMX and leave the dreamConn.asp as it is

- If you want to use DSNless:

a. If you know the absolute path of directory where your files are:

MM_dreamConn_STRING = "Driver={Microsoft Access Driver (*.mdb)};DBQ=C:\InetPub\domain\database\dbblogMX.mdb

OR

MM_dreamConn_STRING = "Driver={Microsoft Access Driver (*.mdb)};DBQ=C:\Sites\username\database\dbblogMX.mdb

b. If you don't know the absolute path, use Server.Mappath:

MM_dreamConn_STRING = "DRIVER={Microsoft Access Driver (*.mdb)};DBQ="& Server.MapPath("database\dbblogMX.mdb")

OR if your Hosting requires a username

MM_dreamConn_STRING = "DRIVER={Microsoft Access Driver (*.mdb)};DBQ="& Server.MapPath("username\database\dbblogMX.mdb")

OR your Host requires your domain name:

MM_dreamConn_STRING = "DRIVER={Microsoft Access Driver (*.mdb)};DBQ="& Server.MapPath("domain\database\dbblogMX.mdb")

The database folder should have write permissions

D. SETUP BLOG

Login to blog Adminitration (http://localhost/blogMX/cryptadmin on localhost or http://yourdomain.com/blogMX/cryptadmin on server)

Login using username "admin" and password "admin"

Click on change user link. Enter new username and password

Click on blog Configuration. Make necessary changes

You need CDO installed on your server for emailing comments to you.

Include Calendar will include calendar in your blog

Include Months will include month links to archive to your blog

Include Category will include category select box to your blog

Include Search will include search box to your blog
